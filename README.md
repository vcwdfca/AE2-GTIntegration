# AE2-GTIntegration

An addon for Applied Energistics 2 Supergiant and gtceu.

# Features

- Add a button to enable to ignore integration circuits in JEI recipe.
- Add Circuit Card inspired by [NAE2](https://github.com/AE2-UEL/NAE2/blob/2fe8dede709b6917d880f41085cd5ece9042fa7e/src/main/java/co/neeve/nae2/mixin/upgrades/gregcircuit/MixinDualityInterface.java#L52)
